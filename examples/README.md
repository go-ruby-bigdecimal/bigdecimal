# Ruby examples

Pure-Ruby usage of `bigdecimal`, runnable under
[go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo):

```sh
rbgo examples/bigdecimal_usage.rb
```

| File | Shows |
| --- | --- |
| [`bigdecimal_usage.rb`](bigdecimal_usage.rb) | Exact decimal add, `to_s` notation, precise `div`, `**`, `round`/`floor`/`ceil` with rounding modes, `divmod`, conversions, and specials. |
